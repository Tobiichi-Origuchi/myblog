## 1. 什么是 Linux，为什么要用 Linux

{% alert(tip=true) %}
你说的对，但是 Linux 是由 Linus Torvalds 牵头研发的一款全新开源类 Unix 操作系统内核。系统运行在一个被称作「一切皆文件」的底层逻辑中，在这里，被 Root 选中的人将被授予「Sudo 权限」，导引 Shell 指令之力。你将扮演一位名为「系统管理员」的神秘角色，在自由的编译中邂逅构建各异、包管理独特的发行版们，和他们一起击败依赖冲突，找回失散的配置文件。
{% end %}
